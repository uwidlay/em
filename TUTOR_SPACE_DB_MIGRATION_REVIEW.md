## 8. Ожидаемый RLS-подход для будущей миграции

| Таблица / объект | Ожидаемый RLS-подход | Источник |
|---|---|---|
| Все доменные таблицы public schema | `enable row level security`. | SPEC 18; ARCH 7 |
| `tutors` | Authenticated user видит и меняет только профиль, связанный с `auth.uid()`. | ARCH 7.2 |
| `students` | Репетитор видит/создает/обновляет только своих учеников. Ученик не получает прямой широкий доступ. | ARCH 7.2 |
| `useful_links` | Доступ репетитора только через владение учеником. | ARCH 7.2 |
| `lessons` | Доступ репетитора только к урокам своих учеников. | ARCH 7.2 |
| `lesson_materials` | Доступ репетитора только к материалам уроков своих учеников. | ARCH 7.2 |
| `homework_submissions` | Репетитор читает сдачи своих учеников; ученик создает сдачу через безопасный слой по токену. | ARCH 7.2 |
| `homework_submission_photos` | Репетитор читает фото сдач своих учеников; ученик работает через безопасный поток сдачи. | ARCH 7.2, 8 |
| `update_events` | Репетитор видит события своих учеников; ученик отмечает просмотр через безопасный слой. | ARCH 7.2 |
| `storage.objects` | RLS-политики для private bucket `homework-photos`; без публичного bucket. | SPEC 11, 18; ARCH 8 |
